# TP5 - ChatBot Flutter avec Groq API

Application mobile Flutter de type ChatBot utilisant Groq API compatible OpenAI comme alternative à ChatGPT API.

## Fonctionnalités
- Splash Screen
- Login Screen
- Chat Screen
- Envoi des messages
- Réponse automatique via Groq API
- Design moderne
- Gestion du chargement
- Structure organisée

## Configuration
1. Créer une API Key sur console.groq.com
2. Ouvrir le fichier `.env`
3. Remplacer `PASTE_YOUR_GROQ_API_KEY_HERE` par votre clé.

## Commandes
```bash
flutter pub get
flutter run
```
